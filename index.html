Project Idea

Overview

For my project idea, I want to build a daily scheduling technology. The application helps you plan out your day to day life. A work-in-progress name for this application would be named Dayli. Dayli have an interface that let user allocate their plans with date and time, much like a calendar. What sets Dayli apart is that it will use AI technology to make plans and allocate time for important tasks for the user automatically. For example, Dayli can allocate time for the users based on their to-do list for the day. It continually learns from the user behavior to improve itself. Dayli can exist both as a desktop program and a mobile app that synchronize with each other.

Motivation

A long-term, grand idea that I have is to make a technology that can serve as a full-fledged personal assistant, similar to the AI Jarvis in the Marvel Iron Man movies. However, there are many technologies that are essential to Jarvis that is not yet advanced enough or have yet to be invented. That is why for now I want to only focus on developing one function of such a personal assistant: planning out the user’s day. Productivity in both the workplace and private lives is an important parameter for everyone. Everybody wants to have more time to do what is important to them. Dayli can help with that.

Description

On a manual level, Dayli works much like a calendar app. Users can make new plans on a day, set a start time and and end time for the plan, and set the plan to be whether recurring or non-recurring. Each user will make an account, so it is possible for a user to view other users’ calendars or share their calendar with other users. Users can mark which part of their calendar is private and which part is shareable.

On the automation part, Dayli can make plans for the user. It can accomplish this task in many ways. Dayli can integrate itself with many other applications that already exists on a user’s phone or computer, such as emails, to-do lists and even other hardwares. For example, when Dayli integrates with a user’s email address, it can learn how many emails the user has received, learn how many are important based on past behaviors (something that all major email services like Gmail and Outlook has already implemented) and learn which one should be prioritized for replies. With the data, in the morning, Dayli can allocate a time slot for the user to read and reply to emails. Once the user is done dealing with emails, they can report it back to Dayli (by tapping a button that says “I’m done with this task”, for example). It can use this data to learn how much time the user needs each day for emails and adjust in the future to suit each user’s needs. Dayli can also make plans based on the content of the emails. The same can be done for to-do lists. Many people have a daily to-do list that they fill out in the morning or in the previous evening. If they use a list on an app, Dayli can integrate with the app to learn of these tasks. Based on their priorities, Dayli allocate time for these tasks.

Scheduling is not limited to workplace tasks and can extend to the user’s own life e.g. a user may want to allocate a time in their day to spend with their family or significant other. The technology can even work with other hardware, or in other words, Internet of Things. For example, imagine a user uses a smart fridge that know how many eggs there is in the fridge. If the user runs out of eggs, Dayli can make it a plan for them to go the grocery store after work.

The more a user use Dayli, the smarter the app become. If the user consistently uses less time for emails than the time Dayli allocate for them (and report to Dayli as such), for example, Dayli may allocate less time for daily emails in the future and vice versa. If the user continually uses Dayli, the app can learn of the users’ habits. Of course, the user is able to change the plans and time that Dayli have set for them. Do this enough times and Dayli will know to adjust to the users’ preferences. Ultimately, Dayli is all about making the users’ schedule work for them, not the other way around, automatically and without hidden functions.

Tools and Technologies

Dayli can exists as both a mobile app and a web application/desktop application. These exists as separate apps but needs to be synced together with each user’s unique account. The Internet is an essential part. At the minimum level, there exists a backend AI/machine learning algorithm that continually learns the user’s usage of the app. One level up, it needs to be able to connect to other apps, make use of their data and integrate these into its own learning. As a software, it does not need any special hardwares, only those that a user would already own: computers and mobile phones. However, as an expanding technology, it may need advances from other products e.g. a fridge that knows how many eggs you have in it.

Skills Required

As a software product, the project needs software developers, including mobile developers, web developers and AI specialists. In particular, the main point of the app is the automation, therefore a good team of developers with good knowledge of AI is a must. These positions may be more difficult to fill since it is a specialized field and not as popular as other software positions. Another extremely important role, in my opinion, is user experience (UX) designers. As described, the app works the best the more data that the users allow to share with it. Therefore, it is impertinent that the app does not feel invasive to the users and provide a positive experience. Many other potential UX problems exist. A UX designer role is often seen as not as important as some other technical roles, so it may be difficult to fill these roles with excellent team members.

Outcome

Productivity is something that everyone wants to increase. It is important to prioritize tasks. Making plans with a calendar is one of the best ways to achieve this. However, the time spent prioritizing is time spent not actually being productive. The most productive thing a person can do is to simply do things. If Dayli is successful, it will eliminate the need for daily prioritizing; a user’s schedule will just work for them. Moreover, it does not just stop as scheduling app. The success of this app is a gateway to a bigger goal: true personal assistant. From that point, AI assistants could become more widespread and allow for development of AI assistants in other fields, such as in healthcare or commerce, which as of yet are only filled by humans.
